## Ziel

Sammelt die meisten Punkte, indem ihr die originellsten Antworten habt.

Genau, wie bei Stadt, Land, Fluss.

---

## Aufbau

Jedes Crewmitglied braucht eine Tabelle:

In den Zeilen stehen die Namen der Crewmitglieder,
in den Spalten die Kategorien.

---

## Beispiel

| Kollege | Augenfarbe | Geburtstag | Hobby | Lieblingsessen | Maskotchen | Reiseziel | Tick | Wunsch |
| :------ | ------: | ------: | ------: | ------: | ------: | ------: | ------: | ------: |
| Meier | - | - | - |- | - |- | - | - |
| Müller | - | - | - |- | - |- | - | - |
| Schmitz | - | - | - |- | - |- | - | - |
| ... | - | - | - |- | - |- | - | - |

---

## Ablauf

**Jeder gegen jede!**

Füllt so schnell wie möglich die Spalten zu allen Crewmitgliedern (auch euch selbst) aus.

Wer als erstes fertig ist, ruft laut "Stopp". 
Sofort müssen alle die Stifte fallen lassen und dürfen keine Antwort mehr eintragen.

---

# <Emoji symbol="👉" label="GoGoGo"/>
# Und los!

---

## Abschluss

Lest zur Punktevergabe pro Crewmitglied (Zeile) reihum alle Antworten einer Kategorie (Spalte) vor.
Das betroffene Crewmitglied liest natürlich als letztes die eigene Antwort und erhält dafür keine Punkte.

* 5 Punkte für die richtige Antwort, wenn nur einer sie hat.
* 3 Punkte für die richtige Antwort, wenn mehrere sie haben.
* 1 Punkt für eine alternative Antwort, die allgemein akzeptiert wird.

---

## Reflexion

Lasst die Fragen ein wenig auf euch wirken, bevor ihr eure Erfahrung teilt.

<Hint variant='light' icon={['fas','file-edit']}
line="Macht euch ruhig kurz ein paar Notizen."
/>

<Hint variant='light' icon={['fas','microphone']}
line="Achtet darauf, das jeder zu Wort kommt."
/>

---

## Fragen

1. Wie hat sich das angefühlt?
2. Was fandest Du besonders interessant?
3. Was sagt deine Punktzahl über Dich aus?
4. Was nimmst Du aus dieser Übung mit?
